# user-management
user-management spring boot application

## APIs
- Home Page API:
``
"http://localhost:8080/home" or "http://localhost:8080/"
``
- Signup Page API:
``
"http://localhost:8080/signup"
``
- Login Page API:
``
 "http://localhost:8080/login"
``
- UserDetails API:
``
 "http://localhost:8080/{userName}"
``
- Logout API:
``
 "http://localhost:8080/logout"
``
